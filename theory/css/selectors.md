**Селектор** — это код, который указывает на элемент к которому будут применены стили.

```css
селектор {
  свойство: значение;
}
```

**Универсальный селектор `*`** — выбирает все элементы.

```css
* {
  margin: 0;
  padding: 0;
}
```
