**Переменные** — это сущность которая хранит какое-либо значение.

```css
:root {
  --main-color: #1890ff;
  --main-bg: #bfbfbf;
}

.target {
  color: var(--main-color);
}

.block {
  color: var(--main-color);
  background: var(--main-bg);
}
```
