### Настройка окружения

---

Для начала, вам потребуется установить любой редактор кода.

Из списка наиболее распространенных и бесплатных:
- Visual Studio Code
- Atom
- Sublime Text

В редакторе кода нужно разобраться со следующим:
1. Как создаются файлы
2. Как создаются папки
3. Как переключаться между файлами

**Предполагаемое время:** менее часа
