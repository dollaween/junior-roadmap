##### 1. Какой будет вывод?

```javascript
const phrase = 'Winter is coming'
console.log(phrase.charAt(1))
```

1. `W`
2. `i`
3. `undefined`
4. `null`

<details><summary><b>Ответ</b></summary>
<p>

**Ответ: 2**

Метод `charAt(index)` — возвращает указанный символ строки. Счет начинается от `0`. То есть `charAt(0)` — `W`, `charAt(1)` — `i`.

</p>
</details>
