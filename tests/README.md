##### 1. Какой будет вывод?

```javascript
console.log(title)
let title = 'Star Wars'
```

1. `'Star Wars'`
2. `undefined`
3. `null`
4. `ReferenceError`

<details><summary><b>Ответ</b></summary>
<p>

**Ответ: 4**

На момент вывода `title` в `console.log()` переменная еще не объявлена.
