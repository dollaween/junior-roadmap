##### 1. Сложение
Напишите функцию, которая принимает два числа и возвращает результат их сложения.  Также функция должна вывести результат сложения в консоль.

```js
function sum() { /* ... */ }

sum(10, 5)  // 15
sum(7, 16)  // 23
```

<details><summary><b>Решение</b></summary>
<p>

```js
function sum(a, b) {
  const result = a + b
  console.log(result)
  return result
}
```

</p>
</details>
