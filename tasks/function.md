##### 1. Сложение

Напишите функцию, которая возвращает результат сложения трех чисел.

```js
function sum() {
  /* ... */
}

console.log(sum(1, 2, 3))
// 6
```

<details><summary><b>Решение</b></summary>
<p>

```js

function sum(a, b, c) {
  return a + b + c
}

sum(1, 2, 3)
```

</p>
</details>
